### Sorting List Items

Para ordenar las listas podemos utilizar el m칠todo _sort()_ o las funciones incorporadas _sorted()_. El m칠todo _sort()_ reordena los elementos de la lista en orden ascendente y modifica la lista original.   ```py

  lst.sort(reverse=True)    # descendente

  **Example:**

  ```py

  ```

  sorted(): devuelve la lista ordenada sin modificar la lista original
  **Example:**

  ```py

  # Reverse order

  ```
